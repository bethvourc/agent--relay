# Agent Relay Icon Explorations

These SVGs explore a more modern, minimal icon direction for Agent Relay while preserving the existing CLI palette:

- dark surface: `#121212`
- primary stroke/accent: `#FFB000`
- secondary stroke: `#B87A00`
- signal highlight: `#7EE34B`

Files:

- `agent-relay-icon-01-terminal-face.svg`
- `agent-relay-icon-02-relay-link.svg`
- `agent-relay-icon-03-split-dock.svg`
- `agent-relay-icon-04-modular-mask.svg`
- `agent-relay-icon-05-signal-frame.svg`
- `agent-relay-icon-final.svg`

The final icon keeps the strongest traits from the exploration set:

- two rounded “eyes” to preserve the agent identity
- a lower dock/terminal shape to reinforce CLI/terminal context
- a detached signal bar on the right to imply relay / handoff
- balanced geometry that still reads clearly at small sizes
